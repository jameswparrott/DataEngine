# DataEngine
This is a project for designing and building a game engine centered around data. Java Records were introduced to the language in Java 14 and made a stable feature in Java 16. This project aims to take full advantage of this new language feature. The project uses LWJGL (https://www.lwjgl.org).

LWJGL (v3.4.1) Build details:
- Assimp
- GLFW
- OpenAL
- OpenGL
- stb
